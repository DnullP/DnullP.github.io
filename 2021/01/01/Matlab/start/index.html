初识Matlab

学校公选课选了个matlab，开个小专题记录一下

首先Matlab有着和Python一样的交互窗口，然后试着建立变量、矩阵、四则运算。顺便一提，通过按 tab键打开代码补全（我说怎么这么难受）

几个生成特殊矩阵的函数：

  • zero(x,y)
  • ones(x,y)
  • rand(x,y)

matlab里面对单个变量进行计算的函数都可以直接用于矩阵，对矩阵中的全部元素进行某种计算

A = sin(rand(3,3))

转置函数：

A = rand(3,3)
A = transpose(A)
A = A'          %这两种表达都用于转置矩阵

计算逆矩阵：

A = inv(A)

A = A * inv(A)  %A乘上它的逆矩阵得到单位矩阵

改变数据的显示格式：

A = rand(3,3)
format long   %显示很多位小数
A
format short  %显示很少位小数
A

两个矩阵对应元素相乘：

A = rand(3,3)
A = A.*A

以上规则适用于其他计算：

A = rand(3,3)
A = A.^3
A = A./3

Concatenation

连接两个矩阵：

A = rand(3,3)
B = A'
C = [A, B]   %横向连接列
D = [A; B]   %纵向连接行

Complex number

复数：

a = sqrt(-3)  %Just try it
