闲人的ctf记录

CFT的题包含的知识面相当广泛, 多做一做能学到不少东西, 所以在一天最后的时间里, 不妨看几道CTF题, 拓展拓展能力

Basic Linux Lab

这道题目要求登录指定的linux服务器并找到flag, 给出了服务器域名和端口, 所以我们使用ssh指令登录服务器:

ssh -pxxxxx root@xxx.xxx.xxx.xxx

登录之后检查文件目录:

ls -a
cd ~
ls -a
ls -All

几番查找下找到flag.txt文件, vim打开获得内容:

vim flag.txt
